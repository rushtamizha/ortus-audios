# Ortus Audios - Premium AV Solutions

A high-performance, SEO-optimized web platform for **Ortus Audios**, a luxury Home Theatre and Audio Experience Centre based in Karnataka.

## 🚀 Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **SEO:** Dynamic Sitemap, JSON-LD Local Business Schema, and Robots.txt

## 🛠️ Key Features
- **Dynamic Brand Pages:** Dedicated SEO-optimized routes for Denon, Marantz, ELAC, etc.
- **Global Grade UI:** Dark-themed luxury aesthetic with glassmorphism effects.
- **WhatsApp Integration:** Direct-to-consultation CTA buttons.
- **Local SEO:** Optimized for Karnataka/Kammanahalli regional search dominance.
